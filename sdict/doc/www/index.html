Sdict

What is Sdict?

Sdict is simple dictionary application, written in mind to be small and portable. It support StarDict dictionaries and can be run on different range OSes and platforms.

Requirements

There is only one requirement: Tcl/Tk distribution. By depend of your operation system you need to install appropriate Tcl/Tk distribution, inspect following sites:
  • Main Site: http://tcl.tk
  • ActiveTcl: http://activestate.com
  • Evolane: http://evolane.com

Project page

To download latest Sdict version, please, go to Download section. Also, you can find two small dictionary: Russian to English, English to Russian.
